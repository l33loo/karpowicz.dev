# My Personal Webpage

This is the code of my personal webpage (https://www.karpowicz.dev).

## Getting Started

* Set up:
  1. Clone this repo
  2. Run `npm install` from the root directory

* Build:
  1. Run `npm run build` from the root directory

## Stack

* ejs
* SASS
* Javascript
* npm
* Webpack
